Utilisation de la bibliothèque Plotly

Ce projet utilise la bibliothèque Plotly pour créer des graphiques interactifs dans les notebooks Jupyter. Plotly est normalement installé par défaut dans la distribution Anaconda, donc vous ne devriez pas avoir besoin d'installer quoi que ce soit de supplémentaire.

Cependant, si vous ne pouvez pas afficher les graphiques interactifs dans les notebooks Jupyter, vous pouvez ouvrir le fichier HTML correspondant pour les voir. Les fichiers HTML sont générés automatiquement lorsque vous exécutez les notebooks Jupyter contenant des graphiques Plotly.

N'hésitez pas à nous contacter si vous avez des problèmes ou des questions concernant l'utilisation de Plotly dans ce projet.
